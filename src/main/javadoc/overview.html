Celect Java Client Library

Usage:

The Java client library is created using a builder as follows:

String endpoint = "https://examples.celectengine.com:30409";
BasicPDBCredentials credentials = new BasicPDBCredentials("userId", "password");
CelectPDBClient celectPDBClient = new CelectPDBClientBuilder().celectPDBEndPoint(endpoint).withCredentials(credentials).build();

The builder will create the client and authenticate it to the specified pDB instance. The client can then be used to interact with the instance.

For example:

List<Project> projects = celectPDBClient.getProjectsForUser("someUser");
System.out.println("The user has " + projects.size() + " projects.");
